== Description ==

This plugin allows you to twit your newly published / edited wordpress posts. "Post to twitter" form is embedded
into "Add/Edit Post" page, thus you can easily create customized twitter posts. You can also shorten your post's
urls for twitter "on the fly" with http://nn.nf url shortener service.

1.0.6 -- {wnnnf} added for www short links

1.0.5 -- twitter post result notification added

1.0.4 -- asterix password without "save this password?" browser's popup

1.0.3 -- now versions 2.5+ are supported (was 2.7+)

== Installation ==

1. Upload `twitplusnnnf` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
